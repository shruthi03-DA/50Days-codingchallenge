# 🐍 Python Challenges: Data Processing

## 📌 Project Overview
This repository contains solutions to basic Python data processing challenges using **strings and tuples**. These tasks demonstrate fundamental data cleaning, transformation, and extraction techniques commonly used in data analysis.

## 🎯 Objectives
- Perform string manipulation and formatting  
- Handle duplicate and unique values  
- Clean and standardize text data  
- Combine multiple datasets  
- Extract useful information from structured data  

## 🧩 Challenges and Solutions

1️⃣ Student Name Formatting
Description: Convert names into proper case format.
✅ Concept Used: String capitalization (.title() / .capitalize())

2️⃣ Count Total Subjects
Description: Count total number of subjects and unique subjects.
✅ Concept Used: len() and set()

3️⃣ Remove Extra Spaces
Description: Clean unwanted spaces from feedback strings.
✅ Concept Used: .strip()

4️⃣ Combine Name & Course
Description: Combine student names with their respective courses.
✅ Concept Used: zip()

5️⃣ Extract Email Domains
Description: Extract domain names from email addresses.
✅ Concept Used: String splitting (.split('@'))
