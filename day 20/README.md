# Medical Insurance Cost Analysis (SQL Project)

The goal of this project is to analyze a medical insurance dataset using SQL and extract meaningful insights related to insurance costs.
The dataset includes details such as age, gender, BMI, number of children, smoking habits, region, and insurance charges.

Approach Created a structured MySQL table for storing insurance data Inserted sample records for analysis Used SQL concepts such as: 
SELECT, WHERE GROUP BY, ORDER BY CASE statements Aggregate functions (COUNT, AVG) Conditional aggregation.

Solved business-driven queries to uncover patterns and trends Key Analysis Performed Counted total number of insurance records.
