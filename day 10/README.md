# Instagram Engagement & Content Performance Dashboard
Project Overview
This project analyzes Instagram post performance data to understand audience engagement, content effectiveness, and growth trends.

# Dataset Features
Post Type (Image, Video, Reel, Story)
Likes, Comments, Shares, Saves
Reach & Impressions
Follower Count
Dashboard Features
Engagement Trend Analysis
Top Performing Posts
Post Type Comparison
Follower vs Engagement Correlation
Interactive Filters (Post Type, Month)

# Key Insights
Reels generate the highest engagement rate
Caption length impacts engagement moderately
Hashtags do not always guarantee higher engagement
Engagement trends vary monthly
Higher followers slightly improve engagement

# Tools Used
Power BI
DAX
Data Cleaning (Power Query)
