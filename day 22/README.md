# Day 22 - Sales Data Analysis using Python(Variables & Data Types)
# Overview
This project is part of my Data Analyst learning journey.
In this challenge, I analyzed a single e-commerce order using Python variables and data types.

The goal was to understand how basic Python concepts are applied in real-world business calculations such as pricing, discounts, and tax computation.

# Objectives
Store sales data using Python variables
Identify different data types
Perform price calculations
Apply discounts
Calculate GST (18%)
Display formatted billing output

# Given Data
Order ID
Customer Name
Product Price
Quantity
Discount
Membership Status

# Concepts Used
Variables
Data Types (str, int, float, bool)
Arithmetic Operations
Conditional Statements
Output Formatting

# Operations Performed
Declared variables with appropriate data types
Printed variable data types using type()
Calculated total price before discount
Applied discount
Added GST (18%)
Generated final bill output

# Sample Output
Order ID: ORD12345 Customer: Rahul Total Price: 2399.97 Discount Applied: Yes Final Amount (including GST): 2548.77

# Learning Outcome
This project helped me understand how Python fundamentals can be used for business-oriented data processing and financial calculations 
An essential skill for Data Analysts.
