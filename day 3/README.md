# Healthy Kids — Nutrition & Growth Analysis
# Project Overview
This project analyzes the relationship between children's dietary habits, lifestyle behaviors, and physical growth using a sample dataset. The analysis was conducted using Microsoft Excel with PivotTables, PivotCharts, and an interactive dashboard.

# Data Preparation
Converted raw data into structured table format
Calculated BMI for each child
Categorized Health Status based on BMI
Created age group classifications
Analysis Performed
Health status distribution by age group
Relationship between nutrient intake and BMI
Impact of sugary drinks and screen time
Regional differences in diet and growth

# Dashboard
An interactive dashboard was created with slicers for:
Gender
Health Status
Region

# Key Findings
High prevalence of overweight and obesity
Higher sugary drink intake linked to higher BMI
Greater screen time associated with unhealthy weight
Regional differences in nutrition patterns

# Recommendations
Reduce sugary drink consumption
Promote fruit and vegetable intake
Encourage physical activity and reduce screen time

# Tools Used
Microsoft Excel
PivotTables & PivotCharts
Data Visualization
