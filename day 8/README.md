# Logistics & Supply Chain Performance Dashboard
🚀 Project Overview
This project analyzes logistics and supply chain data to evaluate delivery performance, identify delays, and optimize operational efficiency using Power BI.

# 🎯 Objective
Analyze delivery performance across regions and cities
Identify delayed shipments
Monitor delivery time and cost
Provide actionable insights for optimization

# 📁 Dataset
The dataset includes:
Shipment ID
Order, Delivery & Expected Dates
Cities & Regions
Distance & Cost
Delivery Status
Delivery Time (Days)

# 📊 Key Performance Indicators (KPIs)
✅ Total Shipments: 300
✅ On-time Delivery %: 49%
✅ Avg Delivery Time: 5.63 days
✅ Total Delivery Cost: 322.98K USD

# 📈 Dashboard Visuals
📊 KPI Cards
📊 Avg Delivery Time by Region (Bar Chart)
🌍 Delivery Status by City (Heatmap)
📋 Shipment Delay Analysis (Table)
🎯 Interactive Filters (Region, City, Status)

# 📌 Key Insights
⚠️ On-time delivery rate is below 50%, indicating major delays
📍 Some cities like Kolkata & Mumbai show higher shipment volumes
🕒 North region has the highest delivery time
💰 Delivery costs are significant and need optimization
