## Strings & Tuples Coding Challenge

I worked on solving practical problems using Python Strings and Tuples. 
This challenge focuses on tuple operations, string manipulation, and basic data processing logic.

Challenges Covered: Find the longest name in a tuple Join tuple elements into a single formatted string
Reverse each string inside a tuple Count vowels in each name Convert tuple elements into uppercase formatted output
Concepts Practiced: Tuple iteration String slicing join() method max() with key=len Generator expressions Basic data analysis logic

This exercise helped strengthen my understanding of string manipulation and tuple handling in Python, which are essential for real-world data processing tasks.
