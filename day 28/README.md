## Overview
focuses on performing Insurance Claims Data Analysis using core Python concepts. 
This project simulates a real-world scenario where a Data Analyst processes raw insurance claim data stored in lists.

The goal is to practice data filtering, aggregation, conditional logic, and list operations in Python.

## Problem Statement
I am working as a Data Analyst for an insurance company. I received claim information containing:

Policy IDs
Claim Amounts
Claim Status
Customer Age
Using Python lists, multiple analytical tasks are performed to extract meaningful business insights.

## Tasks Completed
✔ Calculate total approved claim amount
✔ Count rejected insurance claims
✔ Identify high-value claims (> ₹10,000)
✔ Compute average age of approved customers
✔ Assign customer risk categories based on age

## Skills Practiced
Python Lists
Looping (for loops)
Conditional Statements
Data Filtering
Aggregation Logic
Basic Data Analysis Thinking

## Learning Outcome
This project strengthens foundational Data Analyst problem-solving skills by transforming raw list-based data into actionable insights.
